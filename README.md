# gamezop-api

Rest Api written in Go

## How to use it
* clone this repo
* `cd gamezop-api`
* `./gamzop -host 127.0.0.1 -port 8080` **remove the leading ./ on windows system**

```
Usage of ./gamezop:
  -host string
        Host (Default is 0.0.0.0) (default "0.0.0.0")
  -port string
        Port (Default is 9090) (default "9090")
```
